Cursos para desarrollo web

Contamos con una gran variedad de profesores dispuestos a ayudar para empezar completamente de cero y terminar convirtiéndote en un desarrollador web profesional, estos cursos están ordenados de lo más básico a lo más complejo, te recomendamos seguir este orden, pero si tu ya eres experimentado sobre el tema puede empezar por el que te resulte más necesario.

HTML

Empezaremos por HTML, lo consideramos el más básico junto con CSS, este nos permitirá maquetar una página web.

CSS

CSS nos permitirá darle estilos a una página web, ya sean colores, bordes, formas, etc. En este caso es necesario que sepas los básico de HTML.

JavaScript

Después de haber terminado los dos anteriores cursos, subimos el nivel, JavaScript es uno de los lenguajes de programación más usado hoy en día, debido a que se puede usar tanto en el Frontend como el Backend.

Angular y TypeScript

Angular es una de los frameworks más potentes y utilizados en el mundo para el Frontend, lo veremos en profundidad además de los conceptos básico de TypeScript .

Firebase

Firebase es un servicio de Google, el cual nos brinda una base de datos ya armada y es muy potente a la hora de manejar mucha información.

Cursos para desarrollo mobile

Contamos con algunos cursos basicos, pero pronto publicaremos más.

Swift

Swift es un lenguaje creado por Apple para el desarrollo de aplicaciones iOS y macOS, es una muy buena base para adentrarte al desarrollo mobile.

Kotlin

Kotlin es por excelencia el mejor lenguaje de programación para desarrollo de aplicaciones Android, aprender Kotlin y Swift te permitirá manejarte en los SO más usados del mundo.

Ionic

Ionic es un framework para el desarrollo mobile, este se ajustara bien para aquellas personas que sepan Angular ya que se pueden combinar, lo cual facilitara el aprendizaje.
